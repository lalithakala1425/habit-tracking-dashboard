# 🧠 Habit Tracking Dashboard

A simple and interactive **Habit Tracking Dashboard** built using **HTML, CSS, and JavaScript**. This project helps users track daily habits, monitor progress, and stay consistent with their goals.

---

## 📌 Features

- ✅ Add new habits  
- 🗑️ Delete habits  
- ✔️ Mark habits as completed  
- 📊 Track daily progress  
- 💾 Data stored in browser (Local Storage)  
- 🎨 Clean and responsive UI  

---

## 🛠️ Technologies Used

- **HTML5** – Structure of the application  
- **CSS3** – Styling and layout  
- **JavaScript (Vanilla JS)** – Functionality and interactivity

1. Download or clone the repository:
   git clone https://github.com/your-username/habit-tracker.git

2. Open the project folder.

3. Double-click on `index.html`  
   OR open it using any browser.

---

## 💡 How It Works

- Users can enter a habit in the input field.  
- Clicking the **Add** button adds it to the list.  
- Each habit can be:
  - Marked as completed ✔️  
  - Removed ❌  
- The app stores data using **Local Storage**, so habits remain even after refreshing the page.
